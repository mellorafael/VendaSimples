# Venda Simples UI
Venda Simples UI é um WebApp é o seu POS Web. Realize vendas de onde estiver.

## Tecnologias
As tecnologias foram utilizadas:

```bash
Vue.JS 2.6.12
Bootstrap 4
vue-routes
```
## Dependências
[Venda Simples API](https://github.com/mellorafael/VendaSimplesApi)

```
npm install
```
